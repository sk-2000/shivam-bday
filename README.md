# 🎂 Shivam Nagpal - The Senior OG Birthday Celebration Website

A premium, highly interactive, 3D single-page celebration website dedicated to **Shivam Nagpal**, the Senior OG and disciplined Head Moderator of the **Diksha gamer YT** YouTube community. 

This project features an isotropic 3D mix-colored theme, floating glassmorphic UI elements, interactive party poppers, and a custom background music player designed to celebrate his journey from 0 subscribers to a thriving community.

---

## ✨ Features

- **Isotropic 3D Theme:** A seamless blend of deep cosmic dark modes with vibrant neon accents, creating visual depth.
- **Floating Parallax Design:** All UI cards and elements dynamically react to cursor movements for a premium 3D feel.
- **Interactive Celebrations:** Continuous 3D ambient particle confetti with explosive 3D party poppers triggered on user clicks.
- **Audio Soundscape:** A floating, interactive music control widget featuring a custom Indian Indie-Electronic/Future Bass birthday anthem (`Crown_the_Senior_OG.mp3`).
- **Autoplay Resilience:** Built-in error handling for modern browser autoplay restrictions (audio initiates strictly upon user interaction).
